# custom-newtab
"Fork" of minimal newtab. Essentially a rewrite with more customizability as the original contains a lot of spaghetti code and has little features.

## what I changed
- wasn't supposed to change everything, but i changed everything.

# installation
### manual installation
clone repo, run `npm i` and `npm run build`, then load `./dist/extension` as an unpacked extension in your browser.
### normal installation
go to releases and click on the file that ends with .crx. Google should prompt you to install from there.
